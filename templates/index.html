Book Recommeder System

This is a book recommender system built on flask using the collaborative filtering algorithm.It suggests books that the users may prefer based on users or items.

Getting Started Source on GitHub

How this system works

  • This system takes user id as an input.
  • based on user : It then find people that similar to the user.
  • based on item : or it then find items that similar to what the user like
  • we alse provide a top10 book ranking list

Issues

  • Right now, the app uses the collaborative filtering algorithm.
  • In the future we would like to add a book-marking system.
  • The item-based algorithm may be slow.
  • It doesn't work on real time data.
